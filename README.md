# Hello World using Servlet

A foundational web application demonstrating the basic request-response cycle using **Java Servlets** and **JSP**.

## 🚀 Overview
This project serves as a starting point for learning Java Web Development. It demonstrates how to map a URL to a Java class and display dynamic content in a browser.

## 🛠️ Features
- **Simple Servlet**: A Java class that handles GET/POST requests.
- **JSP Integration**: A basic frontend to trigger the Servlet.
- **Dynamic Greeting**: Displays a "Hello World" message generated by the server.

## 📂 Project Structure
- `helloJSP.jsp`: The home page containing a link to call the Servlet.
- `Servlet1.java`: The Java class annotated with `@WebServlet` that processes the request.
- `WEB-INF/`: Standard directory for web application configuration.

## ⚙️ How to Run
1. Import this project into **Eclipse** or **Spring Tool Suite**.
2. Right-click the project folder.
3. Select **Run As** > **Run on Server**.
4. Choose your **Tomcat 9** server.
5. The browser will open to: `http://localhost:8080/HelloWorldServlet/`

## 📝 Prerequisites
- **JDK 1.8**
- **Apache Tomcat 9.0**
- **Eclipse/STS IDE**
